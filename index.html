Practica 1 alojamientos Web
1.Busca ventajas e inconvenientes de tipos de alojamientos web
 Hosting gratuito:
 Informacion
 Ventajas
 -Hay sitios profesionales para alojar un sitio sencillo de manera gratuita.
-Puedes guardar archivos en ubicaciones diferentes para ocupar menos espacio.
 Desventajas
 -Es posible que no puedas monetizar tu sitio web
-Espacio de almacenamiento limitado
-Ancho de banda limitado
 Hosting compartido:
 Informacion
 Ventajas
 -El Hosting compartido es más barato en comparación con el Hosting dedicado y el Hosting VPS.
-La gestión y el mantenimiento del servidor es tarea del proveedor de alojamiento
-No es necesario ningún conocimiento técnico especial o avanzado para gestionar un sitio web en un Hosting compartido
-Se accede a varias cuentas de correo electrónico con un dominio propio -Hay soporte para MySQL y PHP
 
Desventajas

-Problemas de seguridad en el servidor ya que suelen ser propensos a ataques de hacking, software malicioso que afectan a todos los sitios alojados en el servidor
-Al compartir los recursos con otros sitios, se experimentan una lentitud en los procesos y la carga del sitio
-Hay limitaciones respecto a la memoria, el espacio en disco y el CPU
-El plan de alojamiento puede tener menos características y funciones en comparación al Hosting dedicado
 Hosting para revendedores:
 Informacion
 Ventajas
 El precio es sin duda la mayor ventaja del hosting reseller. Dichos planes de alojamiento son relativamente baratos,
quizás cuesten unos pocos dólares al mes.  
Desventajas

-En comparación con otras empresas de alojamiento web establecidas, estos mismos distribuidores
no son capaces de proporcionar el correcto y completo apoyo técnico fiable.
 Hosting VPS:
 Informacion
 Ventajas
 -La libertad de un servidor dedicado.
-Administrar varios sitios criminal cuenta única.
-Mejor seguridad.
-Flexibilidad.
-Combinan las características de alojamiento compartido y dedicado.
-Se adapta tanto a medias como a grandes empresas.
 
Desventajas

-Limitación de uso de CPU y RAM
-Ancho de banda es limitado
-El tiempo de procesamiento es mayor
-Cliente tiene que controlar su servidor sino es administrado
 Hosting dedicado:
 Informacion
 Ventajas
 -Exclusividad
-Administración
-Seguridad
-Administración
-Mantenimiento
-Versatilidad
-Sistema operativo
 
Desventajas

-El precio tan alto.
 Hosting en la Nube o Cloud Hosting:
 Informacion
 Ventajas
 -Ahorro de costes
-Confiabilidad
-Manejabilidad
-Implementación de las ultimas tecnologias
 
Desventajas

-Falta de tiempo -Seguridad -Dependencia de un proveedor -Control limitado
